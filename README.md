# üöÄ Hero Apps ‚Äî HERO.IO

HERO.IO is a modern and scalable React-based web application that showcases a collection of apps created by our team. Users can browse, search, and explore different applications with smooth navigation and a clean interface. The platform includes dynamic routing, installation tracking using localStorage, real-time search filtering, and smooth animations.

HERO.IO focuses on user experience, performance, and clean code structure. With responsive layouts, optimized loading states, and a consistent design system, it works seamlessly across all devices. It is a fully SPA-based application, enabling instant page transitions, custom loaders, and rich UI feedback.

---

## üåü Main Features

- Browse all available apps with detailed information  
- Real-time search with loading indicators  
- Install and uninstall apps (stored locally via localStorage)  
- Trending apps section with interactive UI  
- Custom fallback pages for invalid routes or missing app data  
- Fully responsive design for mobile, tablet, and desktop  
- Smooth animations and transitions across pages  

HERO.IO is built with modern development practices, making it easy to extend and maintain. It can be used as a portfolio showcase for applications or as a base for larger projects.

## üíª Local Development Setup

Follow these instructions to run HERO.IO locally.

### **Prerequisites**
- Node.js (v14 or higher)  
- npm (comes with Node.js)  
- Optional: imgbb API Key (for image uploads if needed)

### **Setup Steps**
1. Clone the repository:
git clone https://github.com/jubayer-ahmed-ratul/plateshare_client.git

2. Navigate to the project folder:
cd plateshare_client

3. Install dependencies:
npm install

4. Start the development server:
npm run dev

6. Open your browser at:
[[npm run dev](http://localhost:5173)](http://localhost:5173)
